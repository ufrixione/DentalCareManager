 DentalCare Manager 🦷

# Descripción General del Sistema
*DentalCare Manager* es un sistema ligero de gestión de citas odontológicas diseñado para optimizar la administración de pacientes y horarios en un consultorio dental. 

Originalmente concebido como una aplicación de consola en C#.

# Características Principales:
* Gestión de Pacientes:** Registro detallado incluyendo nombre, datos de contacto y cálculo de edad basado en la fecha de nacimiento.
* Control de Agenda:** Programación de citas con validación de horarios para evitar empalmes.
* Persistencia de Datos:** Creación automática y gestión de la base de datos.
